## Changes

- Replaced Shop sample products, orders, customers, discounts, collections, inventory, and analytics arrays with empty production-safe arrays.
- Kept Shop type definitions and tenant admin navigation.
- Replaced fake KPI values with `0` or honest `未集計` states.
- Replaced visible fake admin account name with `ストア管理者`.
- Removed fixed admin date range from the default topbar state.
- Added empty states for products, orders, inventory, customers, analytics, best sellers, inventory alerts, and content.
- Repaired Shop admin links to `/s/aiboux/admin/...`.
- Repaired storefront links to `/s/aiboux/...`.
- Added `src/pages/shop/[tenant]/[...path].astro` to serve storefront child routes.
- Removed `href="#"` from Shop storefront components.
- Removed old `/shop/products`, `/shop/settings`, and `/shop/storefront` links from Shop UI.
- Removed visible sample SKU placeholder from the product editor.
